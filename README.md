# 教学平台单塔斜拉模型

## 安装
1. 克隆存储库：
    ```bash
    git clone https://github.com/your-repo/TeachingPlatformSinglePylonCableStayedModel.git
    ```
2. 进入项目目录：
    ```bash
    cd TeachingPlatformSinglePylonCableStayedModel
    ```
3. 安装依赖项：
    ```bash
    npm install
    ```

## 使用方法
1. 启动应用程序：
    ```bash
    npm start
    ```
2. 打开浏览器并访问 `http://localhost:5173`。
